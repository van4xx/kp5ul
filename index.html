04  Продвинутое SEO

Пять слоёв технического и смыслового SEO

Не «прописать ключи», а перестроить сайты так, чтобы поисковик понимал, ранжировал и показывал их богатым сниппетом.

1  Техническое SEO

  • Серверный рендер (SSR/SSG) — HTML отдаётся готовым, цель по Core Web Vitals: LCP < 2.5 с, INP < 200 мс, CLS < 0.1.
  • Оптимизация загрузки: next/image (WebP/AVIF, lazy), next/font, критический CSS, code-splitting, preload LCP-изображения.
  • Кэширование и Brotli через Caddy; чиним отключённый кэш на скульптуре (no-store, Expires 1981).
  • robots.txt и sitemap.xml генерируются кодом (у скульптуры сейчас оба — 404).
  • Canonical на каждой странице + 301 www/без-www и http/https на единый хост.
  • Карта 301-редиректов со старых URL Tilda/PHP — сохраняем накопленный вес и позиции.
  • Корректный <html lang="ru"> на обоих (у скульптуры сейчас lang="en").

2  On-page

  • Формула title: услуга + район + город + УТП/бренд («Аренда бьюти-места в Мурино, СПб — от 280 ₽/час»).
  • Осмысленный meta description с ключами и гео вместо слогана «Форма важна».
  • Ровно один H1 и логичная иерархия H2/H3 под кластеры услуг.
  • Развёрнутый уникальный текст на посадочных вместо thin-content.
  • Описательные alt с ключом и гео; осмысленные имена файлов.
  • Внутренняя перелинковка хабами, описательные анкоры, устранение страниц-сирот.
  • Полный Open Graph + Twitter Card (1200×630, og:locale=ru_RU).

3  Семантическое ядро и посадочные

  • Сбор ядра: Wordstat + Keyword Planner + подсказки, чистка минус-словами, группировка по интенту и частотности.
  • Один кластер = одна страница: под каждый тип аренды/процедуру и под связку «услуга + район/метро».
  • Пилар-страницы «Аренда для бьюти-мастеров» и «Коррекция фигуры» + информационные страницы под People-Also-Ask.

4  Микроразметка Schema.org

  • HealthAndBeautyBusiness / MedicalBusiness: адрес, geo, телефон, часы, priceRange, sameAs (соцсети, Яндекс Бизнес, 2ГИС).
  • Service + Offer / OfferCatalog: прайс структурированно (аренда 280–480 ₽/час; процедуры 2 900–8 100 ₽, абонементы до 70 000 ₽).
  • FAQPage — на блоках вопросов-ответов (претендент на rich-сниппет).
  • Review + AggregateRating — только по реальным отзывам, без выдуманных рейтингов.
  • BreadcrumbList — хлебные крошки на всех вложенных страницах.
  • VideoObject / ImageObject — для видео техник и галерей.

5  Локальное SEO

  • NAP-консистентность символ-в-символ: сайт ↔ Schema ↔ Яндекс Бизнес ↔ 2ГИС. Устранение опечатки адреса на evebeauty.
  • Гео-запросы в контент и метаданные: Мурино/Девяткино и м. Беговая/Приморский район; гео-блоки «как добраться» с картой.
  • Локальные посадочные «услуга + метро», мониторинг локальной выдачи и наращивание отзывов на картах.
05  Индексация

Google Search Console + Яндекс Вебмастер

Оба сайта заводим, верифицируем и берём под мониторинг — включая корректную работу с IDN-доменом скульптуры (xn--80atdqfckfd1g.com).

Google Search Console

  • Оба ресурса как domain property, подтверждение через DNS TXT (покрывает http/https, www и без-www сразу).
  • Для скульптуры — корректное распознавание IDN по Punycode xn--80atdqfckfd1g.com.
  • Отправка sitemap.xml, проверка отчёта «Индексирование»: все продающие страницы в индексе.
  • Ручная приоритетная индексация ключевых страниц через «Проверку URL».
  • 301 со старых URL, устранение 404, канонизация, весь трафик на HTTPS.
  • Отчёт Core Web Vitals (мобайл/десктоп) и валидность структурированных данных.
  • Email-уведомления о критичных проблемах, передача доступов клиенту.

Яндекс Вебмастер

  • Добавление и верификация обоих сайтов, метод, устойчивый к передеплою.
  • Основное зеркало и склейка дублей www/без-www на скульптуре (301 через Caddy).
  • Отправка sitemap и корректный robots.txt (у скульптуры оба сейчас 404).
  • Региональность: привязка «Санкт-Петербург» (Мурино / м. Беговая) для гео-запросов.
  • Контроль обхода и «Страниц в поиске», ручной переобход важных страниц.
  • Диагностика, валидация Schema.org, мониторинг ИКС «до/после».
  • Связка с карточками Яндекс Бизнеса, Турбо-страницы — как опция.

07  Организации на картах

Перенастройка Яндекс Бизнес и 2ГИС

Единый NAP на сайте и на картах — прямой сигнал доверия для локального ранжирования и главный источник клиентов «рядом со мной».

Яндекс Бизнес

  • Подтверждение владения обеими карточками, полный доступ к правкам и отзывам.
  • Единый NAP как на сайте; устранение разнобоя адреса коворкинга.
  • Точная геометка входа (пин на дверь, этаж, помещение).
  • Рубрики, часы, услуги с ценами = как на сайте.
  • Фото по стандарту Карт + видео техник у студии.
  • Включение и обработка отзывов, публикации и акции.
  • Привязка карточки к Вебмастеру, заполнение «на 100%».

2ГИС

  • Заявка на управление обеими организациями, подтверждение прав.
  • Единый NAP синхронно с Картами и сайтом.
  • Отдельная правка геоточки/входа (свой геокодер 2ГИС).
  • Рубрики из справочника, услуги с ценами.
  • Фото, логотип, у студии — кабинеты и аппаратура.
  • Кнопки действий (сайт, звонок, WhatsApp/Telegram).
  • Включение и сбор отзывов, контроль рейтинга.

Единый стандарт

  • Эталон NAP в одном документе для сайта, Бизнеса и 2ГИС.
  • Schema.org LocalBusiness с теми же данными, что в карточках.
  • Единые геокоординаты входа во всех трёх системах.
